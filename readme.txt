I N S T A L L A T I O N

LINUX (32-BIT & 64-BIT)

	Download Streaker.py to a place of your choosing.

	Streaker requires python 3.x. First, install Python with your package manager.
	
	Then, to run the program, open a terminal and type:
	
		python Streaker.py
	
	If you get an error like "No such file or directory," the python script (Streaker.py) is probably not in your path.  You can try one of the following:
	
		You can try typing the full path of the script, such as:
		
			python /home/user/Streaker.py
	
		You can change directory into the python script's folder and execute the file, such as:
	
			cd /home/user
			python Streaker
	
	If the program still doesn't work, make sure python 3.x is installed.
	
	If the program still doesn't work, open up your file manager, find the python script, right click it, and make sure the script is listed as executable.
	


WINDOWS (2000, XP, VISTA, & 7)

	Download Streaker.py to a place of your choosing.
	
	Streaker requires python 3.x so that must be installed in order for the program to run.  Follow the instructions below for installation.
	
	Download Python from one of the following addresses:
	
		(32-bit) http://www.python.org/ftp/python/3.3.2/python-3.3.2.msi

		(64-bit) http://www.python.org/ftp/python/3.3.2/python-3.3.2.amd64.msi

	Install Python by double-clicking the .msi file and following the onscreen instructions
	
	To run Streaker, double-click the python script (Streaker.py).
	
	If the program doesn't run, the script is probably not in your path.  Trying copying the script to the desktop and double-clicking it.
	
	To run a python script from a location outside of your path, try adding python to your path variable.
	
		In Windows 2000 and XP, do the following:
		
			Right-click My Computer > click Properties > click Advanced tab > click Environment Variables > highlight Path, click Edit
			Add the following to the end of the Variable Value:
			
				;C:\Python26
	
			Click OK

		In Windows Vista and 7, do the following:
		
			Click Start > click Run > in the Run box, type:
			
				accounts
				
			Click Enter > click Change My Environment Variable > highlight Path > click Edit > add the following to the end of the Variable Value:
			
				;C:\Python26
			
			Click OK





U S A G E

	The flash card can have any name you want but it must be in the same folder as streaker.py.

	Questions in the flash card file must start and end with question marks (see flash.txt).

	Answers in the flash card file must start and end with @ symbols (see flash.txt).
	
	Type in the name of your flash card file.
	
	Press enter to see the answer.
	
	Type in a number to determine when the program should show you the card again.
